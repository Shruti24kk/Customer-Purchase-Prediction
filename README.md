# Smart Retail – Customer Purchase Prediction

An end-to-end **machine learning project** that predicts a customer’s **next purchase category** and generates **Top-K recommendations** using synthetic retail data.

---

## Overview

This project simulates a **smart retail environment** and demonstrates a complete ML pipeline:

**Data Generation → Feature Engineering → Customer Segmentation → Purchase Prediction → Recommendations**

The system helps retailers understand customer behavior and make **data-driven marketing decisions**.

---

## Key Features

- Synthetic retail dataset (customers, products, transactions)
- Exploratory Data Analysis (**EDA**)
- Feature engineering using **RFM metrics**
- Customer segmentation with **K-Means clustering**
- Next purchase category prediction using **Random Forest**
- Top-K category recommendations with confidence scores
- Predictions exported to **CSV**

---

## Tech Stack

- **Python 3**
- **NumPy, Pandas**
- **Matplotlib**
- **Scikit-learn**
